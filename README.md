# Smart Bus Tracking System

## Project Description
The Smart Bus Tracking System is a Spring Boot application that helps manage and track buses in real time.

## Technologies Used
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Postman

## Features
- Add Bus
- View All Buses
- Update Bus Details
- Delete Bus
- REST API Support

## API Endpoints
GET /api/buses
POST /api/buses
PUT /api/buses/{id}
DELETE /api/buses/{id}
